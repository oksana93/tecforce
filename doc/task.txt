Написать web-приложение:
1. Приложение собирает данные о температуре и ветре в трёх городах:
Самара,
Санкт-Петербург,
Москва.
Сбор данных происходит каждый час, собираются текущие данные.
2. Предоставляет интерфейс для просмотра данных.
Экран со списком городов.
Для каждого города отображается текущая погода.
Нажимаем на город, показывается погода за последние 28 дней
В ячейке показывается минимальные и максимальные значения за день для температуры и скорости ветра.

Для получения погоды использовать:
https://developer.yahoo.com/weather/?guccounter=1

Порядок выполнения:
1. Написать каркас.
2. Описанить объектную модель.
3. Сгенерировать тестовые данные.
4. Написать экраны просмотра данных.
5. Написать сбор данных.
6. Добавить красоты: тесты, документация.

Стек: Spring boot, PostgreSQL, React.
Срок: 2 недели, по итогам каждой недели - показывать результат независимо от проделанного объёма. По итогам 2 недель задание может быть продлено.
